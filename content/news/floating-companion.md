# Meet The Oracle — Your Floating Companion on Docs & Academy

**Date:** July 6, 2026
**Author:** Agenthood Team

Reading docs is better with a guide. Starting today, every Agenthood docs page, academy lesson, and getting-started guide includes **The Oracle** — an animated floating companion that helps you learn faster.

## What It Does

The Oracle appears as a floating nerd emoji (🤓) in the bottom-right corner of your screen. Click it to open a chat panel with four quick actions:

- **Test my knowledge** — get quizzed on the page content
- **Summarize this page** — request a concise summary
- **Leave feedback** — share your thoughts about the docs
- **Ask me anything** — type any question

The companion tracks your session (up to 20 messages) and includes a live character counter.

## Where to Find It

The Oracle is available on every page under these routes:

- `/docs/*` — architecture, members, agentic-workflows, governance, portals
- `/academy/*` — all three learning levels
- `/getting-started/`

## Built With

- **React 19** + **Tailwind CSS 4** — zero new dependencies
- CSS keyframe animations for the floating balloon and slide-up panel
- 18 Playwright tests covering desktop and mobile (iPhone 13)

Head over to the [Docs](/docs/members/) page and look for the floating nerd in the corner.
